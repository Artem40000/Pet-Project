# Pet-Project
Pet-Project на Django. 
Pet-проект на Django. Регистрация и авторизация с возможностью изменения и удаления пользователя. Есть REST API (Swagger), которое позволяет: удалять всех/одного пользователя, изменять данные о пользователе, добавлять пользователя, выводить список всех/одного пользователя. База данных не вписана.


WeatherProject на Django.
Погода(Температура и Скорость ветра) с локациями, информация с открытого API (OpenMeteo). Доступные локации: Россия: Москва, Казань | Америка: Нью-Йорк, Чикаго.
