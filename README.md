# Pet-Project
Pet-Project на Django. 
Pet-проект на Django. Регистрация и авторизация с возможностью изменения и удаления пользователя. Есть REST API (Swagger), которое позволяет: удалять всех/одного пользователя, изменять данные о пользователе, добавлять пользователя, выводить список всех/одного пользователя. База данных не встроена.
